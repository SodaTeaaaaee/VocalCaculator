## 3. 初始分类建议

以下只作为初始建议，不是最终运行时规则。
最终运行时规则优先写入受版本控制的 Rust 硬编码表；如果后续确有必要外置，才允许改成受版本控制的元数据文件。

### 3.1 高可信语义候选

1. `四` 系列 => `DigitFour`
2. `五` 系列 => `DigitFive`
3. `二` 系列 => `DigitTwo`
4. `三` 系列 => `DigitThree`
5. `除♂` => `OperatorDivide`
6. `方根` / `平方根♂` => `FunctionSqrt`

### 3.2 先归入噪声池

1. `卟卟卟`
2. `卟卟`
3. `呃`
4. `哧`
5. `eng`
6. `噗`
7. `啊↑`
8. `啊→`
9. `U♂`
10. `于`
11. `卟零`
12. `卟零♂`

## 4. 外部音乐资源事实

本仓库当前未内置 `AR7778-digitized-MIDI` 的资源。

若需要开发期接入，必须使用受控外部路径并记录：

1. 上游 URL
2. commit hash
3. 本地资产路径
4. 是否仅本地开发使用
